## 快捷键

| 按键           | 说明        | 
|--------------|-----------|
| 箭头 ↑         | 时钟字体增大    | 
| 箭头 ↓         | 时钟字体减小    | 
| 箭头 ←         | 毛玻璃层模糊度减小 | 
| 箭头 →         | 毛玻璃层模糊度增大 | 
| <kbd>C</kbd> | 变换颜色      |
| <kbd>D</kbd> | 日期 显示/隐藏  |
| <kbd>S</kbd> | 秒数 显示/隐藏  |
| <kbd>H</kbd> | 帮助 显示/隐藏  |
| <kbd>G</kbd> | 毛玻璃效果切换   |
| <kbd>F</kbd> | 全屏切换      |
